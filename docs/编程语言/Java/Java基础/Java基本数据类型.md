# Java基本数据类型

> 作者：梁帝  
> 编者按：基本数据类型是在编程中最容易被忽视的技术点之一。他们配合相应的包装类，往往发挥着极其重要的作用，最简单的一个例子是：定义一个类的属性时，我应该用其包装类吗？

Java共有8种基本数据类型

```
整型：byte、short、int、long
字符型：char
浮点型：float、double 布尔型：boolean
```

### 1. byte
- 8位有符号的数据类型
- 包装类是Byte
- 最大值：Byte.MAX_VALUE 127
- 最小值：Byte.MIN_VALUE -128
